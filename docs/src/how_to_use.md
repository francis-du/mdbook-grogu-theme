# How to use

- Init

run `mdbook init`

- Config book.toml

```shell
[output.html]
theme = "grogu"
default-theme = "grogu"
```

- Clone theme

```shell
mkdir theme && cd theme

git clone https://github.com/francis-du/mdbook-grogu-theme grogu
```

- [Live demo](https://grogu-theme.francis.run)

```shell
https://grogu-theme.francis.run
```
